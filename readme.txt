== Description ==
A quick, easy way to add and display WordPresss Recent Post Widget with 3 Designs.

View [DEMO](http://demo.wponlinesupport.com/recent-posts-widget-designer-demo/) for more details.

This plugin is based on the well-known WordPress default widget 'Recent Posts' and extended to display more informations about the posts like e.g. thumbnails, date, design, author, excerpts and assigned categories.

= Designs =

* Design-1 : Left side thumbnail and right side content. 
* Design-2 : First thumbnail and then content. 
* Design-3 : First Row - First thumbnail and then content and next all Rows : Left side thumbnail and right side content.

= Use Following Widget Options =

* Title of the widget
* Number of listed posts
* Design for widget (3 designs given)
* Random order of posts
* Show post author
* Show post categories
* Show post date
* Show post excerpt
* Show posts of selected category (or of all categories)


== Installation ==

1. Upload the 'recent-posts-widget-designer' folder to the '/wp-content/plugins/' directory.
2. Activate the "recent-posts-widget-designer" list plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance --> Widget section and check for Designer - Recent Post Widget
